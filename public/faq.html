JuiceBox FAQ

Everything you usually ask before uploading.

Basics

What is JuiceBox?

JuiceBox is a minimal temporary file host. You upload a file (max 500MB) and instantly get a direct link. Files auto‑expire after the retention you select (1 hour – 14 days).

Do I need an account?

No. Ownership is associated with the IP address that uploaded the file (Cloudflare edge IP only retained internally for rate limiting; end‑user IP is not stored).

What is the file size limit?

Each file can be up to 500MB. Folders dropped in the UI are zipped client‑side first; the resulting ZIP must also fit the 500MB cap.

Which file types are blocked?

Potentially dangerous executables / scripts (e.g. .exe .dll .bat .cmd .com .js .ps1 .sh .php etc.) are rejected to reduce abuse.

Can I hotlink files?

Yes. Links returned ( /f/<id> ) are direct and cacheable by browsers/CDN while still expiring server‑side.

Retention & Expiration

How is expiration determined?

A UNIX timestamp is stored at upload time: now + chosen_ttl_seconds. A background task plus on‑access checks remove expired metadata and disk files.

Can I extend a file's life?

Not currently. Re‑upload the file with a longer retention.

What happens after expiration?

The metadata entry is dropped and the file is deleted from disk. The link returns 404.

Privacy & Security

Do you log my actual IP?

The service runs behind Cloudflare. Only the Cloudflare edge IP (not the true end‑user IP) is persisted for ownership mapping. CF headers exposing the visitor IP are ignored.

Are uploads encrypted?

In transit: yes (HTTPS). At rest: files are stored unencrypted on the server's filesystem. Do not upload sensitive personal data.

Do you scan malware?

No automated malware scanning right now. Executable/script extensions are blocked by name only. Treat downloaded files cautiously.

Can other people list my files?

No. Listings are IP‑scoped. Without your exact link a stranger cannot enumerate your files.

Abuse & Limits

Rate limits?

A simple token bucket (per edge IP) allows bursts up to 60 requests refilling at 1 token/second.

Content rules?

No illegal content, malware, personal data you lack rights to share, or anything that violates applicable laws. See Terms of Service.

Removal

How do I delete a file early?

Use the UI delete button (❌) or send DELETE /d/<filename> from the same network that uploaded it.

Can staff remove files?

Yes. Admin intervention (e.g. abuse reports) can purge content immediately; your listing will reconcile automatically.

API Notes

Is there an API?

Unofficial: POST /upload (multipart form with one or more file fields and an optional ttl field) returns JSON: { files: ["f/abc123..."] }.

Retention field values?

1h 3h 12h 1d 3d 7d 14d. Invalid values default to 3d.
